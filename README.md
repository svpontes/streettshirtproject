# Street T-shirt E-Commerce of Exclusive T-Shirts

## About the Project
**street t-shirt** is a web application developed as part of a group project for the course (cse325 .NET Software Development). The main objective is to create a functional e-commerce platform specializing in the sale of t-shirts with exclusive high-definition (PNG) designs.

This project serves as a key assessment to demonstrate our skills in the .NET development ecosystem, teamwork, and the application of good web design practices (performance, accessibility, and usability).

## Technology Stack
* **Framework:** .NET Blazor (WebAssembly / Server)
* **Language:** C#
* **Frontend:** HTML5, CSS3 (with responsive design practices) or tailwind css
* **Version Control:** Git & GitHub
* **Project Management:** Trello

## Main Features
To meet the project requirements, our application includes:
1. **Catalog** 1. Product Management:** Responsive display of t-shirts cards with images, prices, and descriptions.

2. User Authentication:** Secure registration and login for customers and administrators.

3. CRUD Operations:** *Administrators:* Create, read, update, and delete products (t-shirts) and manage inventory.

*Customers:* Add items to the shopping cart and manage their addresses/orders e personal info.

4. Shopping Cart and Checkout:** Complete online purchase simulation flow.
